# Sanatlium Twitter Client

## 概要
TitaniumでTwitterクライアント制作。
主にPythonで書かれています。

## 依存関係
- oauth2
- httplib2
- simplejson
- tweepy
- jQuery1.4.1

## 使用法
自分の環境でしか確認してないのけど、走らせるのは多分面倒。
運が良ければSanatlium.pyのconsumerとtokenをきちんと設定すれば走ります。
